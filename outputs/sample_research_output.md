## Response

Explainable AI (XAI) aims to make AI systems understandable to the humans who use them. For novice users who lack technical knowledge of machine learning, this challenge is especially important.

### 1. Transparency

The system should make its reasoning visible. Users should be able to see which input features most influenced an AI decision [Ribeiro et al., 2016]. Tools like LIME and SHAP provide feature importance scores that can be presented in plain language.

### 2. Interpretability Through Simplicity

Explanations should use concepts familiar to the user, avoiding technical jargon. Research shows that example-based explanations (showing similar past cases) are often more intuitive for novices than feature-weight diagrams [Cai et al., 2019].

### 3. Interactivity

Allowing users to ask "why?" questions and explore counterfactuals (e.g., "What would change this decision?") significantly improves user understanding and trust [Wachter et al., 2017].

### 4. Trust Calibration

Overly confident AI outputs can mislead novices. Effective XAI communicates uncertainty honestly, helping users know when to rely on or override AI recommendations [Zhang et al., 2020].

### 5. Context-Aware Explanations

Different users in different contexts need different levels of detail. Adaptive explanation depth that matches user expertise improves satisfaction and task performance [Lim et al., 2009].
